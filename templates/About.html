About This Project

This is my first backend web development project â€” a ToDo List app made using Flask (Python web framework).

The project allows users to:

  • Add new tasks with a name and hobbies (used as task description).
  • View all tasks on the home page.
  • Edit or update existing tasks.
  • Delete tasks when they are no longer needed.

The app uses an SQLite database to store the tasks. For managing the database, I used Flask-SQLAlchemy ORM.

The interface is simple and user-friendly. It includes a form for adding new tasks and displays the task list in a table format. Users can click "Update" to edit a task or "Delete" to remove it.

Technologies used in this project:

  • Python (Flask)
  • HTML (Frontend templates)
  • CSS (for styling)
  • SQLite (Database)
  • Jinja2 (Template rendering in Flask)

This project helped me understand the basics of backend development like:

  • Routing and URL handling in Flask
  • Form submission and validation
  • Database integration using models
  • Flash messages and redirects

I plan to improve it further by adding user login/signup, task deadlines, and priority features in the future.
